# Simba Supermarket Rebuild

A modern, fast, and beautiful e-commerce experience for Rwanda's leading supermarket.

## 🚀 Features
- **Modern Tech Stack:** Built with Next.js 14, Tailwind CSS, and Zustand.
- **Multilingual:** Support for English, French, and Kinyarwanda.
- **Dark Mode:** Premium dark theme support.
- **Product Discovery:** Instant search, category filtering, and product detail pages.
- **Mobile-First:** Optimized for mobile users with an app-like feel.
- **Checkout Flow:** Simulated checkout with Mobile Money (MoMo) integration.
- **Animations:** Smooth interactions powered by Framer Motion.

## 🛠️ How to Run Locally
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (Navbar, Cart, ProductCard, etc.).
- `src/store`: State management with Zustand (Cart, Theme, Language).
- `src/lib`: Data utilities, translations, and helper functions.
- `src/types`: TypeScript definitions.

## 🎨 Design System
- **Gold (#FFD700):** Symbolizing quality and value.
- **Deep Blue (#0054A6):** Representing trust and stability.
- **Red (#ED1C24):** Used for urgent actions like "Out of Stock" or cart counts.
